# Gym Website Design System

## Design Style

Modern Fitness Brand

Keywords:

* Professional
* Premium
* Bold
* Energetic
* Clean
* Modern

---

# Color Palette

Primary:

* #E63946

Background:

* #111111

Secondary Background:

* #1A1A1A

Text:

* #FFFFFF

Muted Text:

* #BDBDBD

Borders:

* #2A2A2A

---

# Typography

Font Family:

* Poppins
* Inter
* Sans-serif fallback

Hierarchy:

H1

* Large
* Bold
* Strong visual impact

H2

* Section heading

Body

* Clean and readable

---

# Buttons

Primary Button

* Red background
* White text
* Rounded corners
* Hover animation

Secondary Button

* Transparent
* White border
* Hover effect

---

# Cards

Used For:

* Features
* Services
* Trainers
* Testimonials
* Pricing

Requirements:

* Rounded corners
* Soft shadows
* Hover lift effect
* Consistent spacing

---

# Layout

Desktop:

* Multi-column layouts

Tablet:

* Reduced columns

Mobile:

* Single-column layouts

---

# Spacing

Section Padding:

* 80px desktop
* 60px tablet
* 40px mobile

Container Width:

* Max width 1200px

Center aligned content

---

# Animations

Use lightweight animations only:

* Fade-in effects
* Hover transitions
* Smooth scrolling

Avoid heavy animation libraries.

---

# Images

Use placeholder image URLs.

Do NOT generate AI images.

Use placeholders for:

* Hero section
* Trainers
* Gallery

Images will be manually replaced later.

---

# Responsiveness

Website must work on:

* Mobile
* Tablet
* Laptop
* Desktop

Navigation must collapse into hamburger menu on mobile.

---

# Final Goal

The website should look like a professional gym business website that could be sold directly to a local fitness center with minimal modifications.
